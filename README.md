# FleetSync
## Unified Ride-Hailing & Logistics Platform for Saudi Arabia

FleetSync is a comprehensive platform designed to integrate ride-hailing (RideSync) and logistics (LogiSync) into a single, efficient ecosystem.

### Features
- **Unified Dashboard**: Real-time overview of all operations.
- **Live Tracking**: Monitor vehicles and shipments across Saudi Arabia.
- **RideSync**: Passenger transport management.
- **LogiSync**: Freight and logistics management.
- **Analytics**: Data-driven insights for optimization.

### Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Properties), Vanilla JavaScript
- **Design**: Modern Dark UI, Glassmorphism, Responsive Layout
- **Font**: Inter (Google Fonts)

### Getting Started
1. Clone the repository.
2. Open `index.html` in your browser.
3. No build step required!

### Deployment
This project is ready for immediate deployment on Vercel, Netlify, or GitHub Pages.
